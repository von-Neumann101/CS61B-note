# 拓扑排序
![[Pasted image 20260410102720.png|313]]
v$\to$w代表在w之前，必须先有v（比如E，必须先做BDF才行）
对于这一棵有向树，我们需要给定一个能执行的顺序，如ACBDFEHG
一个不错的想法：每次移除所有没有被指向的顶点(root)，但是时间复杂度是O(EV)，利用空间换时间，我们用数组追踪每个顶点，可以优化一点，但不够

我们对每个入度为0的顶点运行DFS，**而且不清除mark**（其实可以从任意顶点开始）
